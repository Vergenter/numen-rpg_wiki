# Umiejętność/Fechtunek

**Fechtunek** - umiejętność dodająca +1 statystyki w walce w sposób
klasyczny.

Nie działa na:

- atakujących w magiczny sposób
- zwierzęta

Dodatkowo fechtunek usuwa karę -1 statystyki ataku przy walce bronią
białą.

## Wariacje

- Siła
- Atak
- Unik
- Blok
- Zręczność
- Szybkość
