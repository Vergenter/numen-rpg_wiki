# Umiejętność/Obrona sojusznika

**Obrona sojusznika** - umiejętność, dzięki której wybrany sojusznik
dostaje połowę wartości statystki bloku użytkownika pod warunkiem
trafienia blokowanego przeciwnika, postać dalej normalnie może wykonywać
atak, ale k160.

Dodatkowo obrona sojusznika daje +1 do statystyki ataku do osłaniania
lub bloku(wariacja).

Obostrzenia:

- umiejętność nie działa na ataki dystansowe
- obie postacie muszą znajdować się blisko siebie oraz przećwiczyć
  odpowiedni szyk
- działa na jednego przeciwnika
