# Runa

**Runa** - Metoda ułożenia Visu, powodująca konkretne [zjawisko
magiczne](zjawisko-magiczne.md).

Rozmiar run jest wprost proporcjonalny do siły. Większe runy dużo
łatwiej zdestabilizować.

Do destabilizacji mogą doprowadzić np.:

- [wiatry magiczne](wiatry_magiczne "wikilink")
- efekty innych run
- i inne zjawiska magiczne
