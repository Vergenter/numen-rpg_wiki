# Runa wystrzału

**Runa wystrzału** - runa magiczna służąca do przenoszenia do
konkretnego punktu słowa runicznego.

## Działanie

W tym wariancie zaklęciu jest nadawany pęd magiczny, przez co łatwiej
nim celować i szybciej leci, ale takie zaklęcie nie zawiera pędu
rzeczywistego.

### Parametry

- Zasięg: 8 / 16 / 24 \[yd\]
- Celność: +10/+10/+20 strzelectwa
- Szybkość: prędkość strzały z łuku
- Koszt: 1 / 2 / 2 \[erg\]
