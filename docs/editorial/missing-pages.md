## Braki zasad gry

| Brak | Dlaczego potrzebny |
| --- | --- |
| Tworzenie postaci | Są karta postaci, rasa, statystyki i umiejętności, ale nie ma kolejności tworzenia postaci. |
| Awans postaci | Drzewko umiejętności opisuje strukturę rozwoju, ale brakuje kosztów, limitów i momentu awansu. |
| Walka | Są rany, atak, strzelectwo i obrona sojusznika, ale brakuje pełnej sekwencji tury walki. |
| Ruch i dystans | Potrzebne do walki, pościgów, zasięgów run i przeszkód. |
| Ekwipunek i ceny | Brakuje wspólnej listy broni, pancerzy, narzędzi, kosztów i dostępności. |
| Leczenie i odpoczynek | Są rany, ale brakuje gojenia, pierwszej pomocy, odpoczynku i trwałych skutków. |
| Statusy | Brakuje wspólnych reguł dla podpalenia, zimna, bólu, strachu, zmęczenia, trucizn i ogłuszenia. |

## Braki magii i run

| Brak | Dlaczego potrzebny |
| --- | --- |
| Gramatyka run | Są pojedyncze runy i słowo runiczne, ale brakuje zasad składania run w poprawny zapis. |
| Katalog run | Runa ognia, lodu, fali, stali i inne powinny mieć wspólną tabelę kosztów, efektów i ograniczeń. |
| Błędy czarowania | Jest `Czarowanie`, ale brakuje skutków nieudanego testu, przeciążenia i ryzyka dla użytkownika. |
| Przestrzeń magiczna w praktyce | Istnieje definicja, ale brakuje reguł użycia przy testach, runach i zjawiskach. |
| Zjawiska magiczne | Strona istnieje, ale powinna zebrać typy zjawisk, skalę, czas trwania i konsekwencje. |

## Braki świata

| Brak | Dlaczego potrzebny |
| --- | --- |
| Oś czasu | Import ma wprowadzenie i część pojęć, ale brakuje chronologii er i wydarzeń. |
| Geografia | Brakuje regionów, miast, granic, zagrożeń i map. |
| Frakcje | Brakuje stron organizacji, państw, kultów, szkół runicznych i ich celów. |
| Rasy poza człowiekiem | Jest `Rasa` i `Człowiek`, ale brakuje innych ras albo jasnego stwierdzenia, że ich nie ma. |
| Bestiariusz | Brakuje stworzeń, ich zachowania, statystyk i roli w świecie. |

## Braki porządkowe

| Brak | Dlaczego potrzebny |
| --- | --- |
| Słownik pojęć | Trzeba zebrać `Vis`, `Ko`, runy, przestrzeń magiczną, statystyki i umiejętności w jeden indeks. |
| Plan połączenia importu | Trzeba zdecydować, które strony z importu są kanonem, które są stare, a które trzeba przepisać. |
| Kategorie | Zaimportowane kategorie istnieją jako strony, ale nie są jeszcze użyte do porządkowania treści. |
