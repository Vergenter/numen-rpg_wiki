# Kategoria

**Kategoria** miejsce Definicja standardu tworzenia kategorii.

Każda kategoria powinna składać się z opisu i wzoru artykułów w niej
zawartych.

Jeśli żaden wzór nie został zdefiniowany to obowiązującym jest poniższy

## Wzór

**Pogrubiona \[Nazwa strony\]** oraz definicja słowa będącego nazwą
artykułu, nie używając żadnego specjalnego stylu do tego.
