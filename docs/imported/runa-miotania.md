# Runa miotania

**Runa miotania** - runa magiczna służąca do przenoszenia do konkretnego
punktu słowa runicznego.

## Działanie

Ten wariant został stworzony specjalnie, by móc nadać zaklęciu pęd
rzeczywisty i takie użycie runy wymaga zamachu w tym testu strzelectwa.
Kształt jak i waga pocisku dobierane są tak, żeby uzyskać jak najszybszą
prędkość rzutu.

### Parametry

- Zasięg: rzutu
- Celność: +10/+10/+20 strzelectwa
- Szybkość: prędkość rzucania, czyli zauważalnie wolniej od łuku, ale
  raczej nadal szybko
- Koszt: 1 / 2 / 2 \[erg\]
