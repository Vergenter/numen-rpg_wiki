# Umiejętność/Walka bronią

**Walka bronią** - umiejętność zwiększająca jedną z poniższych statystyk
przy użyciu wybranej broni.

Dodatkowo umiejętność:

- usuwa karę -1 ataku do danego typu broni
- udostępnia specjalne użycia broni

Dostępne bronie:

- miecze
- topory
- obuchy
- broń długa

## Wariacje

- Siła
- Atak
- Blok
- Unik
